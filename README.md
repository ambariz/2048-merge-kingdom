# Merge Kingdom

A medieval kingdom building puzzle game inspired by 2048.  
Merge matching structures to grow your civilization from a humble tent into a legendary empire or galaxy ;)

[![Click to Play](https://img.shields.io/badge/CLICK%20ME%20TO%20PLAY-itch.io-red?style=for-the-badge&logo=itchdotio)](https://ambariz.itch.io/)

**LINK :** https://ambariz.itch.io/

---

## Gameplay

- Start with small kingdom tiles placed on a 4×4 board
- Use the Arrow Keys to move all buildings
- Merge matching buildings to create stronger structures
- Every move spawns a new building tile
- Continue expanding your kingdom through multiple eras
- Reach the legendary 2048 Kingdom tile to achieve victory

Every run becomes a challenge of planning, positioning, and long-term strategy.

---

## Features

- Classic 2048-style gameplay
- Medieval kingdom progression system
- Custom artwork for every building stage
- Sound effects for merges and milestones
- Victory and Game Over screens
- Move counter and score tracking
- Best score saved using Local Storage
- Smooth tile animations
